# Installation Scripts

This repository includes bash scripts to install Jenkins, Nexus Repository Manager, and SonarQube on Linux servers.

The scripts are written for specific Linux distributions:

- **Jenkins** installation script is designed for **Ubuntu**.
- **Nexus Repository Manager** installation script is designed for **Amazon Linux**.
- **SonarQube** installation script supports **both Ubuntu and Amazon Linux**.

These scripts automate the installation and configuration process for each tool on the specified operating systems.
you can install it on your server.
